Deep learning skin condition prediction model


GitHub

I developed a deep learning model that categorizes skin images into 20 distinct skin disease categories. The model was trained using a Convolutional Neural Network (CNN) to efficiently classify the images based on patterns and features in the dataset. I preprocessed the data using Pandas and NumPy, visualized it with Matplotlib, and built the model using TensorFlow. To enhance scalability, I deployed the model on Azure for faster processing and storage. This project not only helped me improve my skills in deep learning but also demonstrated the potential of AI in healthcare.

Technologies: Python (TensorFlow, Pandas, NumPy, Matplotlib), Azure

GitHub
